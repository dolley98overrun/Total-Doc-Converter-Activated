# Total Doc Converter

## Overview

Total Doc Converter is a software tool designed to convert documents between various file formats. It facilitates the transformation of files such as DOC, DOCX, RTF, TXT, and others into formats suitable for different applications and workflows. The project aims to provide a straightforward solution for document format conversion on Windows systems.

## Key Features

**Supports multiple input formats including DOC, DOCX, RTF, and TXT**  
**Converts documents into formats like PDF, HTML, TXT, and image files**  
**Batch conversion capabilities for processing multiple files simultaneously**  
**Preserves document formatting and layout during conversion**  
**Handles embedded images and tables within documents**  
**Command-line interface available for automation and scripting**  
**Option to customize output settings such as file naming and page range**
